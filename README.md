# Easy Visual

## 安装

Easy Visual (easyvis)是一个基于 d3.js的可视化工具包, 使用SVG进行绘制，内置了常用的图形，使用简单。

通过 npm 进行安装 :

```bash
npm install --save easyvis
```

## 使用

1. 暂无全量文档，请参考IDE工具里面的提示，有全量参数说明。
2. 请参考DEMO [https://vis.zone/easyvis/](https://vis.zone/easyvis/) 。
